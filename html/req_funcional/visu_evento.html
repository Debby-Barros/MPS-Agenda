Visualizar evento

O usuário poderá visualizar os eventos que estão contidos em um dia específico do calendário.

Nome: Visualizar evento
Ator: Usuário
Resumo: Descreve a sequência de funções realizadas para visualizar um evento na agenda.
Pré-condição: Existir login
Pós-condição: Acesso aos eventos da agenda e as funcionalidades existentes nele.
Fluxo principal: 1. abrir o aplicativo da agenda
2. fornecer os dados para o campo: nome e senha
3. clicar em entrar
4. na tela de calendário, clicar no dia que possui um evento para visualiza-lo.
4. Inserir as informações de nome do evento, horário, descrição.
Fluxo alternativo: Realizar cadastro para logar na agenda.

Fluxo de telas

Esse fluxo representa uma sequência de passos de como visualizamos um evento no sistema de agenda.
