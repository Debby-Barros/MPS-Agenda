Cadastro

O usuário poderá se cadastrar utilizando seu nome, email e senha para ter acesso ao login da agenda.

Nome: Cadastro
Ator: Usuário
Resumo: Descreve a sequência de funções realizadas para fazer o cadastro na agenda.
Pré-condição: Existir email para cadastrar.
Pós-condição: Acesso a agenda e as funcionalidades existentes nela.
Fluxo principal: 1.abrir o aplicativo da agenda
2.fornecer os dados para o campo: nome, email, senha e data de aniversário
3.clicar em registrar.
Fluxo alternativo: Se não possuir email, será necessário criar para poder se cadastrar.
