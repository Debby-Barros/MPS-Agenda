Editar dados do usuário

O usuário poderá alterar suas informações pessoais como nome, email, data de nascimento ou alterar a senha.

Nome: Editar usuário
Ator: Usuário
Resumo: Descreve a sequência de funções realizadas para editar os dados do usuário na agenda.
Pré-condição: Existir cadastro
Pós-condição: Acesso ao perfil do usuário e as funcionalidades existentes nele.
Fluxo principal: 1.abrir o aplicativo da agenda
2.fornecer os dados para o campo: nome e senha
3.clicar em entrar.
4. na tela de calendário, clicar no menu sanduíche e escolher a opção de editar perfil ou alterar a senha
Fluxo alternativo: Realizar cadastro para logar na agenda.
