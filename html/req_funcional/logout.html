Logout

O usuário poderá se deslogar da agenda caso queira acessar com outra conta ou apenas sair do sistema.

Nome: Logout
Ator: Usuário
Resumo: Descreve a sequência de funções realizadas para se deslogar da agenda.
Pré-condição: Existir cadastro
Pós-condição: Acesso a agenda para realizar o logout.
Fluxo principal: 1.abrir o aplicativo da agenda
2.fornecer os dados para o campo: nome e senha
3.clicar em entrar.
4. na tela de calendário, clicar no botão "saída" para se deslogar.
Fluxo alternativo: Realizar cadastro para logar na agenda.
