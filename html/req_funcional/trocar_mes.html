Trocar mês

O usuário poderá visualizar os meses do ano através de um botão que terá a funcionalidade de trocar o mês.

Nome: Trocar mês
Ator: Usuário
Resumo: Descreve a sequência de funções realizadas para entrar na tela principal da agenda para trocar o mês no calendário.
Pré-condição: Existir login
Pós-condição: Acesso a agenda e as funcionalidades existentes nela.
Fluxo principal: 1. abrir o aplicativo da agenda
2. fornecer os dados para o campo: nome e senha
3. clicar em entrar
4. no calendário deve se utilizar os botões “<” ou “>” para navegar pelos meses.
Fluxo alternativo: Realizar cadastro para logar na agenda.
