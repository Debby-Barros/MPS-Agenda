Dicionário de dados: agenda_usuario

Trata dos dados referentes ao usuário, armazena e gerencia essas informações.

Nome Tipo Descrição Tamanho
usuario_id INT Número de identificação dos usuários.
usario_nome VARCHAR Nome informado pelo usuário. 100
usario_email VARCHAR Email informado pelo usuário. 100
usario_senha VARCHAR Senha informada pelo usuário. 10
usario_status INT Verificar se o usuário está ou não ativo.

Dicionário de dados: agenda_eventos

Trata dos dados referentes aos eventos da agenda, armazena e gerencia essas informações

Nome Tipo Descrição Tamanho
evento_id INT Número de identificação dos eventos.
evento_data_hora DATETIME Data e hora do evento informado pelo usuário.
evento_descricao VARCHAR Descrever a respeito do evento informado pelo usuário. 200
evento_nome VARCHAR Nome do evento informado pelo usuário. 45
evento_status INT Verificar se o evento está ou não ativo.
usuario_id INT Chave estrangeira que identifica a quem o evento pertence.
